Google Security Reward Programs

Google has enjoyed a long and close relationship with the the security community. Part of this relationship involves providing cash rewards for quality security research that identifies security vulnerabilities in products that we provide or proactive security improvements to select open-source products.

Google currently offers the following security reward programs:

  • Google Vulnerability Reward Program (VRP)
  • Patch Reward Program
  • Vulnerability Research Grants
  • Chrome Reward Program
  • Android Reward Program
  • Google Play Security Reward Program
